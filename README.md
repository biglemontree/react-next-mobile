## Contributing

### Set up

Install dev deps after git clone the repo.

```bash
$ npm i
```

### Init Icon

```bash
$ npm run init
```

### Build Component

```bash
$ npm run compile
or use watch
$ npm run compile --w
```

### Start Dev

```bash
$ npm start
```
